# Syntax

- story-beats uses markdown chapters
- ** camera descriptions uses double stars** **shot of docs face**
- *actions use one start* *picks up pen*
- empty spaces (32) lines can be used to remove short chapter markers
- <!-- commented out line, this text is no longer part of the story -->
- [NUMBER] (3 digit hex number) are used to refer to things in the text, like for instance when something on the checklist gets implemented it could connect the text to the checklist entry.
- pipe | is used for when characters get cut off EX: that was a|
